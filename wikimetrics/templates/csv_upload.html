Format

Wikimetrics expects either a file or pasted text in comma separated format, as shown below. Project is the langauge or project code in which that username or userid exists. Project is optional and will default to the cohort's default project. 

user_name,project
user_id,project


For Example
Specify EITHER user names: 
Evan (WMF),en
Milimetric,en
Without project is ok too
User Names can even contain commas, but these need a project at the end,en
OR User Ids: 
123,en
234,en
456
